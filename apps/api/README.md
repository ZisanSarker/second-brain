# AI Second Brain — API Service

NestJS 11 backend monolith for AI Second Brain.

See the [root README](../../README.md) for installation, deployment, and usage instructions.

## Quick Reference

```bash
pnpm dev              # Start in watch mode
pnpm build            # Compile to dist/
pnpm test             # Run unit tests
pnpm test:e2e         # Run e2e tests
pnpm prisma:migrate   # Apply database migrations
pnpm prisma:studio    # Open Prisma Studio
```

Swagger docs available at `/api/docs` in development mode.
